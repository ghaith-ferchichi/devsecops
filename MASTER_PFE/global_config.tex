\frenchAbstract{%
Ce projet présente la conception et le déploiement du BTE Security AI Agent, un moteur DevSecOps autonome piloté par l'intelligence artificielle locale. Le système intercepte les événements de Pull Request GitHub via webhook (validé HMAC-SHA256), orchestre en parallèle cinq outils d'analyse de sécurité (Trivy, Gitleaks, Semgrep, Checkov, OSV-Scanner) et produit une revue de sécurité complète (OWASP Top 10, score de risque, verdict) en moins de 7 minutes. L'inférence LLM s'exécute entièrement en local via Ollama (\texttt{qwen2.5-coder:7b/14b}) sans aucune exfiltration de code. La plateforme comprend 12 conteneurs Docker, un assistant de chat opérationnel avec 20 outils de monitoring, et une stack d'observabilité complète (Prometheus, Grafana, AlertManager, VictoriaMetrics, node-exporter) avec 12 règles d'alerte.%
}

%%% EN
\englishAbstract{%
This project presents the design and deployment of the BTE Security AI Agent, an autonomous DevSecOps engine powered by local AI inference. The system intercepts GitHub Pull Request webhook events (HMAC-SHA256 validated), orchestrates five security scanning tools in parallel (Trivy, Gitleaks, Semgrep, Checkov, OSV-Scanner), and produces a full security review (OWASP Top 10, risk score, verdict) in under 7 minutes. LLM inference runs entirely on-premise via Ollama (\texttt{qwen2.5-coder:7b/14b}) — no code or diffs leave the infrastructure. The platform includes 12 Docker containers, an operational chat assistant with 20 live monitoring tools, and a full observability stack (Prometheus, Grafana, AlertManager, VictoriaMetrics, node-exporter) with 12 alert rules.%
}
